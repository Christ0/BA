Prototypischer Forward+ Renderer in Vulkan

Da das Vulkan SDK zu groß ist, einfach bei https://vulkan.lunarg.com/ für das jeweilige OS herunterladen.
Mein SDK ist auf Version 1.1.101.0, der Renderer sollte aber auch auf der neuesten Version laufen. 
Wenn nicht, dann mal mit 1.1.101.0 versuchen.


Damit der Renderer funktioniert, einfach das SDK in denselben Ordner kopieren, wo sich auch diese .txt Datei befindet.
Alternativ, in VS die Includeverzeichnisse und Linkereinstellungen ändern.

Ich nutze VS2017 und Windows 7, eventuell müssen bei anderen Versionen die Windows SDK Version und das Plattformtoolset in Visual Studio geändert werden.
